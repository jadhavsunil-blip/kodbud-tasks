# IPL Exploratory Data Analysis (EDA)

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on the Indian Premier League (IPL) dataset to uncover insights about teams, players, venues, and match outcomes using Python.

## 🎯 Objective
Analyze IPL match and ball-by-ball data to identify trends, player performances, team statistics, and venue analysis through data visualization.

## 📂 Dataset
- `matches.csv`
- `deliveries.csv`

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📊 Project Workflow
1. Import libraries
2. Load datasets
3. Data cleaning
4. Check missing values
5. Perform Exploratory Data Analysis (EDA)
6. Create visualizations
7. Draw insights and conclusions

## 📈 Visualizations
- Top 10 Winning Teams
- Top 10 Run Scorers
- Top 10 Wicket Takers
- Top 10 IPL Venues
- Toss Decision Analysis
- Player of the Match Analysis
- Matches Played Per Season

## 📌 Key Insights
- Identified the teams with the highest number of IPL victories.
- Found the top run scorers across IPL seasons.
- Analyzed the leading wicket-taking bowlers.
- Identified the most frequently used IPL venues.
- Studied toss decision patterns (Bat or Field).
- Found players with the most Player of the Match awards.
- Compared the number of matches played each season.

## 🚀 How to Run

1. Clone the repository.

2. Install the required libraries:

```bash
pip install pandas numpy matplotlib seaborn
```

3. Place the following files in the project folder:
- `matches.csv`
- `deliveries.csv`

4. Open `IPL_EDA.ipynb` in Jupyter Notebook or VS Code.

5. Run all notebook cells.
