# Titanic Survival Prediction

## 📌 Project Overview
This project predicts whether a passenger survived the Titanic disaster using Machine Learning. The dataset was preprocessed, explored, and used to train a Logistic Regression model for binary classification.

## 🎯 Objective
Build a machine learning model to predict passenger survival based on features such as age, gender, passenger class, fare, and family information.

## 📂 Dataset
- Dataset: Titanic Dataset
- File: `Titanic-Dataset.csv`

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## 📊 Project Workflow
1. Import libraries
2. Load the dataset
3. Data cleaning and preprocessing
4. Handle missing values
5. Encode categorical variables
6. Split data into training and testing sets

9. Visualize results
10. Draw conclusions

## 🤖 Machine Learning Model
- Logistic Regression

## 📋 Model Performance

- Accuracy: **81.01%**

### Classification Report

| Metric | Not Survived | Survived |
|---------|-------------:|----------:|
| Precision | 0.83 | 0.79 |
| Recall | 0.86 | 0.74 |
| F1-Score | 0.84 | 0.76 |

## 📌 Results
- Achieved **81.01%** prediction accuracy.
- Female passengers had a higher survival rate than male passengers.
- First-class passengers had better survival chances.
- Age and passenger class influenced survival probability.
